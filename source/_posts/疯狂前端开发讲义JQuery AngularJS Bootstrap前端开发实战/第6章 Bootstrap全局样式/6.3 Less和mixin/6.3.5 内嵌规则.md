# 6.3.5 内嵌规则 #
通过`Less`的内嵌规则可以批量地将选择器直接放入指定命名空间下。例如如下代码:
```less
#container{
    .a {
        width:200px;
        height:35px;
    }
    span {
        font-size: 16px;
        text-decoration: none;
    }
    div li {
        list-style: circle;
    }
}
```
<!--replace:li=L I-->
上面的代码先将`.a`、`span`、`div li` 这三个选择器都放在`#container`命名空间下,这表明以`#container`为包来生成包含选择器。上面代码经编译后会生成如下`CSS`样式:
```css
#container .a {
  width: 200px;
  height: 35px;
}
#container span {
  font-size: 16px;
  text-decoration: none;
}
#container div li {
  list-style: circle;
}
```
