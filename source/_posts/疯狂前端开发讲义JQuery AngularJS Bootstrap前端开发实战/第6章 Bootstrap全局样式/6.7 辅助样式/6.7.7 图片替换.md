# 6.7.7 图片替换 #
`Bootstrap`提供了一个`.text-hide`样式用于**隐藏元素中的文本**.
<!--SSTStop-->

其实该样式的源码非常简单:
```css
.text-hide{
    font: 0/0 a;
    color: transparent;
    text-shadow: none;
    background-color: transparent;
    border: 0;
}
```
